# Lotería Number Tracker

This is a simple web page to keep track of sold lottery numbers.

## How to use
1. Open `index.html` in your browser.
2. Click on any number to mark it as sold.
3. Click again to unmark it.

## Files
- `index.html` — main page
- `styles.css` — styles
- `script.js` — logic for clicking numbers

## Notes
You can change the number range inside `script.js`.
